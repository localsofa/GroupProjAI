# Savable Webscraper with UI
Linguistic Computation Groupwork (It scrapes, it gathers...)

# Current Features:
* working simple UI
* user adds URL
* upon button click -> scrapes given URL for title, main content, and list items
* user can choose to save file to specific directory & create folders within in UI
* summarizer of a given URL with option to summarize according to a keyword
* user can click "encode" to (shocker) encode summary
